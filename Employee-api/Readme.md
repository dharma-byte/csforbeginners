# How to load these API's on postman or Bruno

1.Manually Create API Requests

2.Open a workspace.

3.Click "New Request".

4.Select method (GET, POST, PATCH, etc.).

5.Enter the API URL.

6.Add headers, params, or body.

7.Click "Send" to test.

8.It will generate response.
